Instrucciones de uso:
*	Los modelos entrenados se encuentran en la carpeta “modelos”, los de Keras tienen formato .h5 y los del support vector
	machine .sav.

*	Para la ejecución del programa debe ejecutar el comando:
	python3 numberclassifier.py

*	El sistema mostrará una interfaz gráfica de usuario donde podrá seleccionar un modelo haciendo uso del menú desplegable a 
	la derecha del botón clasifica. 

*	Puede dibujar un dígito usando el mouse en la sección blanca de la aplicación.

*	Puede limpiar la pantalla de dibujo usando el botón clear, o puede borrar una parte del dígito usando el eraser. Además
	puede cambiar el tamaño del trazo con el slider a la derecha del botón eraser.

*	Para clasificar oprima el botón “clasificar”.

*	Adicionalmente, el programa permite la oportunidad para entrenar un nuevo modelo de support vector machine en la parte
	inferior donde puede modificar los parámetros y iniciar el entrenamiento con el botón “train”.
